# University Event Management System

A Java desktop application for managing university events. The system allows users to add, view, edit, delete, and search events using a simple graphical user interface built with Java Swing.

## Project Overview

This project is designed to help manage university-related events in an organized way. Users can enter event details such as the event name, date, and description. The application stores event records in a text file so that the data can be loaded again when the program runs.

## Features

* Add new university events
* View saved events in a list
* Edit existing event details
* Delete events that are no longer needed
* Search events by date
* Save event information to a local text file
* Load event records from the saved file
* Simple Java Swing user interface

## Technologies Used

* Java
* Java Swing
* File handling
* NetBeans IDE
* Apache Ant build system
